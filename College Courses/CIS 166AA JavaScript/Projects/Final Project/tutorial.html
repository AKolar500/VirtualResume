Tutorial

Since the first assignment was the most fun for me, I'll give you a tutorial on how to do the first assignment. I'm just going to limit the tutorial to the .js file, since we can assume we all know how to code the html and css for the page.

-----------------------------------------------------------------------

The first thing we need to do is initiate the variables inside the calculate_click method. Copy and paste the following code:

var length = parseFloat( $("length").value );

var width = parseFloat( $("width").value );

Next, we need to calculate the area. The following code will do just that.

var area = length * width;

I like to make the last two digits visible, so you can omit the following code if you don't want it.

area = parseFloat( area.toFixed(2) );

Next, we need to calculate the perimeter.

var perimeter = 2 * (length + width);

Like I said, I like to make the last two digits visible, so if you want to do the same, just copy and paste the previous section of optional code and replace every instance of the word, "area" with "perimeter".

Finally, we need to display the results in their appropriate fields. Copy and paste these last bits of code into the last section of your calculate_click function.

$("area").value = area;

$("perimeter").value = perimeter;

And you're done!
